# 🐍 Navisworks IronPython Reader

**Navisworks IronPython Reader** es un plugin para **Autodesk Navisworks Manage** que integra un entorno de ejecución de scripts en **IronPython**, permitiendo automatizar tareas directamente dentro del entorno de Navisworks.

---

## 🚀 Instalación

1. Descarga el instalador más reciente para la versión **2024** y **2026** desde la sección de *Releases*:  
   👉 [**Descargar última versión**](https://github.com/rafa2403nunez-droid/Navisworks.IronPythonReader/releases)

2. Ejecuta el instalador (`Navisworks.IronPythonReaderSetup.exe`) y sigue los pasos del asistente.

3. Abre **Autodesk Navisworks Manage** y encontrarás una nueva pestaña en el Ribbon llamada **"Iron Python"** con los comandos integrados.

> ⚠️ **Importante:** Para el correcto funcionamiento del plugin, debes tener instalada la versión **IronPython 2.7.12**.  
> Puedes descargarla desde el sitio oficial:  
> 🔗 [https://ironpython.net/download/](https://ironpython.net/download/)

---

## 🧩 Características

- Ejecución directa de scripts **IronPython** en Navisworks.  
- Integración completa con la **API de Navisworks**.  
- Instalador automático generado con **Inno Setup**.  
- Código **abierto y extensible** para desarrolladores.

---

## 🛠️ Uso básico

1. Abre **Navisworks Manage 2024**.  
2. Dirígete a la pestaña **Iron Python Reader**.  
3. Haz clic en el botón **Reader** para ejecutar tu comando.  
4. Personaliza o amplía los comandos modificando los scripts del proyecto.
