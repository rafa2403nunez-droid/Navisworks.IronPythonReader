# 🐍 Navisworks IronPython Reader

**Navisworks IronPython Reader** is a plugin for **Autodesk Navisworks Manage** that integrates an **IronPython** scripting runtime, allowing you to automate tasks directly inside the Navisworks environment.

---

## 🚀 Installation

1. Download the latest installer for **Navisworks Manage 2024** and **2026** from the *Releases* section:  
   https://github.com/rafa2403nunez-droid/Navisworks.IronPythonReader/releases

2. Run the installer (`IronPython.Reader_Navisworks_Version_1.0.1_Installer.exe`) and follow the setup wizard.

3. Open **Autodesk Navisworks Manage**. You will find a new Ribbon tab called **Iron Python** with the integrated commands.

> ⚠️ **Important:**  
> For the plugin to work correctly, **IronPython 2.7.12** must be installed.  
> You can download it from the official website:  
> https://ironpython.net/download/

---

## 🌍 Language Support (Ribbon)

This add-in works in **both English and Spanish versions of Navisworks**.

- Ribbon UI available in **English** (`en-US`)
- Ribbon UI available in **Spanish** (`es-ES`)

No additional configuration is required.  
The plugin automatically adapts to the Navisworks UI language.

---

## 🧩 Features

- Direct execution of **IronPython** scripts inside Navisworks  
- Full integration with the **Navisworks API**  
- Automatic installer built with **Inno Setup**  
- **Open-source and extensible** for developers  

---

## 🛠️ Basic Usage

1. Open **Navisworks Manage 2024 or 2026**  
2. Go to the **Iron Python** Ribbon tab  
3. Click the **Reader** button to run the command  
4. Customize or extend commands by modifying the project scripts  

---

## 💻 Manual Build (Optional)

To build the plugin manually:

- Clone the repository  
- Open the solution in Visual Studio  
- Compile the project for the desired Navisworks version
